# Task Manager Application

A full-stack Task Manager application built using **React, Node.js, Express, PostgreSQL, and Prisma**.
Users can register, login, and manage their tasks with search, filter, and pagination features.

## 🚀 Features

* User Authentication (Register & Login)
* Create Tasks
* Update Task Status
* Delete Tasks
* Search Tasks
* Filter Tasks (Completed / Pending)
* Pagination Support
* Logout System

## 🛠️ Tech Stack

Frontend

* React.js
* Axios
* CSS

Backend

* Node.js
* Express.js
* Prisma ORM

Database

* PostgreSQL

## ⚙️ Installation

### 1 Clone Repository

```
git clone https://github.com/yourusername/task-manager.git
```

### 2 Install Backend

```
cd backend
npm install
npm run dev
```

### 3 Install Frontend

```
cd client
npm install
npm run dev
```

## 🌐 Application Flow

```
Register → Login → Dashboard → Create Task → Update/Delete Task
```
